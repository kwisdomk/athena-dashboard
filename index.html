Warnings & Cautions

RAM Spec Check: Confirm DDR4-3200 (not 2400 or 2666) before purchasing to ensure dual-channel sync.
Storage Bottleneck: The Dahua SSD is PCIe Gen3. It fits the Gen4 slot perfectly but will max out at Gen3 speeds.
Benchmarking: Run Cinebench R23 baseline BEFORE installing RAM for accurate before/after performance comparison.
Battery Degradation: Battery Care Mode (80% cap) must be set in HP Command Center before anything else — zero cost, immediate benefit.
